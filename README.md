# Food Store React

### Package in Project:
- Swiper

## [See Website](https://hambergerstore.netlify.app/)

##  Run 

Clone the project

```bash

git clone https://github.com/mohammadyousefvand/Food-Store-React.git

```

Go to the project directory

```bash

cd Food-Store-React

```

Install dependencies

```bash

npm install

```

Start the server

```bash

npm start

```
